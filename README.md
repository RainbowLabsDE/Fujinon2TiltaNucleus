# Fujinon2TiltaNucleus

Converts the control signal meant for a Fujinon lens to 3 Tilta Nucleus-N outputs. For in-camera control of unsupported / unmotorized lenses.

## Software

Toolchain: PlatformIO + CubeMX  
MCU: STM32F072C8
